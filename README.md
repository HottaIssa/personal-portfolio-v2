# My Page

This repository contains the source code for my personal portfolio website, "My Page". It showcases my skills, projects, and experience as a developer.

## Features

- Responsive design with a clean and minimalistic aesthetic
- Sections for my profile, projects, and experience
- Navigation menu and contact information

## Technologies Used

- Astro: A modern front-end framework for building static websites
- Tailwind CSS: A utility-first CSS framework for rapid UI development
- JavaScript: For interactivity and dynamic behavior
- HTML: For structuring the content of the website

## Installation

To run this project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/my-page.git`
2. Install dependencies: `pnpm install`
3. Start the development server: `pnpm dev`

## Usage

Once the development server is running, you can view the website at `http://localhost:4321`.
